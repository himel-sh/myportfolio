## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

### Build

```bash
npm run build
```

## Customization

### Update Your Information

1. **Hero Section** (`src/components/Hero.jsx`): Update your name, bio, location, and profile image
2. **About Section** (`src/components/About.jsx`): Update your story and background
3. **Skills Section** (`src/components/Skills.jsx`): Replace placeholder logos with actual skill logos
4. **Work Section** (`src/components/Work.jsx`): Add your projects with images and descriptions
5. **Contact Section** (`src/components/Contact.jsx`): Update your email and phone number
6. **Footer** (`src/components/Footer.jsx`): Update your name

### Replace Images

- Replace all `https://via.placeholder.com` URLs with your actual images
- Update skill logos in the Skills component with actual technology logos
- Add your profile photos in Hero and About sections

### Update Social Links

Update the `href="#"` attributes in:

- Header component (navigation links)
- Hero component (social media icons)
- Contact component (social media icons)
